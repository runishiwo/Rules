# 🚀 My Rules & Configs

本项目用于备份 **Mihomo (Clash)**、**Nikki** 以及其他常用软件的个人配置文件。通过 GitHub 进行版本管理，方便在不同设备（如 iKuai、OpenWrt、移动端）之间快速同步。

## 📂 目录结构说明

| 文件夹 | 功能描述 |
| :--- | :--- |
| `yaml` | **Mihomo / Nikki** 的核心配置文件备份 |
| `rule` | 自定义分流规则集（Rule Sets） |
| `ikuai-bypass` | **iKuai** 旁路辅助相关脚本或配置 |
| `sub` | 订阅转换或节点相关信息备份 |
| `zashboard` | 仪表盘、WebUI 面板的自定义配置 |
